# Handy Note
An Android Application for writing notes with special UI and managing notes handy

( Compatible with android 6 or above )

This application implemented using :
- MVVM Architecture
- Room Database

## Download
You can download the APK file from [release](https://github.com/Rez79Kh/Handy-Note/releases/tag/v1.0.0) section.
